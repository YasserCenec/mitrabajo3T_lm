Acerca de este proyecto

Este proyecto ha sido realizado por el alumno Yasser, del curso 1º DAM.

Aquí se han cumplido todas las obligaciones indicadas en la rúbrica del proyecto, incluyendo:

  • Uso de menú persistente en todas las páginas
  • Tres páginas como mínimo
  • Uso de web storage para guardar ajustes entre sesiones
  • Uso de elementos XML y XSL
  • Uso de JavaScript para modificar contenidos de la página
  • Uso de elementos HTML de ingreso de datos
  • Nombre del usuario siempre visible en una de las esquinas del viewport
